# 模型原始清单（去重 + 初步分类）

> 数据来源：`model/Model_Library_Combined.md` + `model/Model_Library_Pro_2026.md`  
> 目标：为“单模型结构化拆解”提供**去重后的模型实体清单**与初步分类。

## A. 优化与规划类（Optimization）

- 线性规划（LP）
- 混合整数线性规划（MILP）/ 整数规划 / 0-1 规划
- 非线性规划（NLP，含 SLSQP 等约束优化）
- 凸优化建模框架（CVXPY）
- 动态规划（DP）
- 多目标优化 / 多目标规划（帕累托前沿；NSGA-II/NSGA-III）
- 加权和法（多目标转单目标）

## B. 智能启发式优化（Heuristics / Metaheuristics）

- 遗传算法（GA）
- 粒子群优化（PSO）
- 蚁群算法（ACO）
- 模拟退火（SA；含 `dual_annealing` / `basinhopping` 等实现路径）
- 人工蜂群算法（ABC）

## C. 评价与决策类（Evaluation & Decision）

- 层次分析法（AHP）
- 熵权法（EWM / Entropy Weight）
- TOPSIS
- 熵权-TOPSIS（Entropy-TOPSIS）
- 灰色-TOPSIS（Grey-TOPSIS）
- 模糊综合评价（FCE）
- 灰色关联分析（GRA）
- 数据包络分析（DEA：CCR / BCC）
- PCA-DEA（降维 + 效率评价融合）
- 博弈论组合赋权（Game Theory Combination Weighting）

## D. 预测与机器学习（Prediction & ML）

### D1. 时间序列/预测
- ARIMA / SARIMA
- 指数平滑（Holt / Holt-Winters）
- Prophet
- 灰色预测 GM(1,1)
- 新陈代谢灰色模型（Metabolic GM(1,1)）
- Logistic 增长模型（S 形增长）
- 马尔可夫链（Markov Chain）

### D2. 回归/分类/集成
- 线性/岭/套索/弹性网回归（回归分析家族）
- 支持向量机（SVR / SVM）
- 随机森林（Random Forest）
- XGBoost / LightGBM / CatBoost
- 高斯过程回归（GPR）
- 神经网络（MLP/CNN/LSTM/GRU 等：按任务选型）

## E. 微分方程与动力系统（Dynamics）

- 常微分方程（ODE）数值解（`odeint` / `solve_ivp`）
- 传染病模型（SI/SIS/SIR/SEIR；含 SEIR-SD 等改进）
- 种群模型（Malthus / Logistic / Lotka-Volterra）
- 偏微分方程（PDE）数值解
- 有限差分法（FDM）
- 有限元法（FEM / FEniCS 等）
- 物理信息神经网络（PINNs：PDE/ODE + NN 融合）
- 差分方程（离散动力学）
- 混沌系统（Logistic map / Lorenz / Rössler）

## F. 图论与复杂网络（Graph & Network）

- 最短路径：Dijkstra / A*
- 全源最短路：Floyd
- 最小生成树（MST）
- 最大流（Max Flow）/ 网络流
- TSP（旅行商问题：常用 GA/SA/DP 求解）
- 社交网络分析（SNA）
- 中心性指标（Degree / Betweenness / Closeness / Eigenvector）
- 社区发现（Louvain）
- 网络鲁棒性分析（随机攻击 vs 蓄意攻击）
- PageRank

## G. 统计与多元分析（Statistics & Multivariate）

- 主成分分析（PCA）
- 非线性降维可视化（t-SNE / UMAP）
- 聚类（K-Means / 层次聚类 / DBSCAN）
- 相关分析（Pearson / Spearman）
- 典型相关分析（CCA）
- 方差分析（ANOVA）
- 判别分析（LDA / QDA）
- 因子分析（FA）
- 结构方程模型（SEM）
- 关联规则（Apriori）

## H. 概率与随机过程（Stochastic）

- 蒙特卡罗模拟（Monte Carlo）
- 排队论（M/M/1, M/M/c, M/G/1 等）
- 生存分析（Kaplan–Meier, Cox）

## I. 仿真建模（Simulation）

- 元胞自动机（CA；如交通流 NS 模型）
- Agent-Based Modeling（ABM / 智能体建模）
- 系统动力学（System Dynamics：库存-流量-反馈）
- 离散事件仿真（DES：SimPy）

## J. 拟合与插值（Fitting & Interpolation）

- 曲线拟合（`curve_fit` / `polyfit`）
- 插值（`interp1d` / `griddata` / RBF）
- 样条插值（Spline：`UnivariateSpline` / `splrep`-`splev`）

## K. 模型检验与灵敏度分析（Validation）

- 灵敏度分析（局部 ±5%/±10%）
- 全局灵敏度（Morris / Sobol）
- 交叉验证（k-fold CV）
