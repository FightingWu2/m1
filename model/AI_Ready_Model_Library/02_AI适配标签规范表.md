# AI 适配标签规范表（受控词表 + 使用规则）

> 本规范用于：  
> 1) 单模型结构化卡片的标签统一；2) 题型-模型匹配矩阵的检索一致性；3) AI 自动匹配的“触发关键词”稳定命中。  
> 原则：**短、准、可枚举、可组合、无歧义**（禁止“可能/大概/通常”）。

## 1) 标签写法（统一格式）

- **模型卡片头部格式**（必须）  
  `# {模型名}_{模型类别}_{适配题型}`  
  示例：`# 遗传算法GA_优化类_MCM-B/ICM-D`
- **标签块格式**（必须）  
  - `【模型类别】...`
  - `【题型适配】...`
  - `【任务类型】...`
  - `【数据特征】...`
  - `【触发关键词】...`
  - `【创新方向】...`

## 2) 维度一：模型类别（必选 1）

仅允许下列 8 类（用于矩阵列、模型检索）：

- 【优化类】（规划、组合优化、多目标）
- 【预测类】（时间序列、回归、机器学习预测）
- 【评价类】（赋权、排序、效率评价、模糊评价）
- 【统计分析类】（统计推断、降维、聚类、相关、结构方程）
- 【图论网络类】（路径、流、中心性、社区、鲁棒性）
- 【动态仿真类】（ODE/PDE/差分/系统动力学/ABM/CA）
- 【随机过程类】（随机模拟、马尔可夫、排队、生存）
- 【拟合插值类】（曲线拟合、插值、样条）

## 3) 维度二：美赛题型适配（可多选）

统一术语（在任何文档中保持一致）：

- 【MCM-A】连续型 / 机理建模 / 微分方程常见
- 【MCM-B】离散型 / 组合优化 / 图论常见
- 【MCM-C】数据洞察 / 统计学习 / 评价常见
- 【ICM-D】运筹优化 / 系统设计 / 资源配置
- 【ICM-E】环境科学 / 生态-扩散-风险
- 【ICM-F】政策分析 / 社会系统 / 决策评估
- 【全题型通用】（当模型本身不依赖题型）

## 4) 维度三：任务类型（可多选）

用于把题目“需求句”映射到模型（矩阵与AI匹配核心）。

- 【排序评估】（排名、择优、优劣比较）
- 【赋权建模】（构建权重、指标体系）
- 【资源分配】（预算、产能、供需、配给）
- 【路径规划】（最短路、TSP、车辆路径）
- 【选址调度】（排班、布点、调度）
- 【预测回归】（趋势、下一年、预测值）
- 【分类分群】（聚类/分类）
- 【因果/关系】（相关、因子、结构路径）
- 【动力演化】（传播、扩散、竞争、演化）
- 【风险不确定】（风险评估、区间、随机性）
- 【仿真复现】（规则复杂、需模拟）
- 【参数反演/校准】（拟合参数、最小二乘/优化）

## 5) 维度四：数据特征（可多选）

- 【小样本数据】（\(n<20\) 或有效样本很少）
- 【中样本数据】（\(20 \le n < 500\)）
- 【大样本数据】（\(n \ge 500\)）
- 【时序数据】（有时间顺序/季节性）
- 【高维数据】（特征多、共线性明显）
- 【无监督数据】（无标签）
- 【缺失数据】（缺失值明显）
- 【噪声数据】（波动大/异常点多）
- 【网络数据】（图结构、节点-边）
- 【空间数据】（含经纬度/网格/区域）
- 【离散决策变量】（0-1/整数）
- 【连续决策变量】（实数）

## 6) 维度五：创新方向（必选 1-3）

只允许下列“可落地”创新标签：

- 【模型融合】（预测+优化、聚类+评价、机理+数据）
- 【参数改进】（正则化、注意力、改进温度/交叉概率等）
- 【鲁棒/不确定性】（蒙特卡罗、区间、稳健优化）
- 【多目标与帕累托】（NSGA-II、帕累托前沿+偏好）
- 【验证与可解释】（灵敏度、消融、特征重要性、置信区间）
- 【约束工程化】（把现实规则显式写进约束/奖励）
- 【可迁移叙事】（把工业/工程方法迁移到社会/环境题并给出映射）

## 7) 维度六：触发关键词（必选 5-12 个）

写法要求：
- 必须是“题面常用词/评审常用词”
- 每个模型卡片至少覆盖：**任务词 + 数据词 + 约束词**（若适用）

示例词表（可复用）：
- 趋势、预测、下一年、季节性、缺失、波动
- 排名、评分、优选、权重、指标体系
- 成本最小、收益最大、约束、预算、容量、调度、选址
- 最短路径、路径规划、运输、物流、TSP、网络流
- 传播、扩散、传染、竞争、演化、稳定性
- 风险、概率、不确定性、置信区间、随机模拟

## 8) 兼容“英文/缩写”规则

- 模型名首次出现：**中文（英文/缩写）**，之后统一用缩写。  
  示例：遗传算法（GA）→ 后文统一 GA。
- Python 库写法：`库名.模块.函数`，不写长代码，只写**可执行入口**。
