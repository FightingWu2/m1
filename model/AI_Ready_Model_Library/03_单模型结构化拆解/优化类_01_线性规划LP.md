# 线性规划（LP）_优化类_MCM-B/ICM-D

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-B，ICM-D
- 【任务类型】资源分配，选址调度（线性），成本最小/收益最大
- 【数据特征】连续决策变量，中大规模约束
- 【触发关键词】线性约束、成本最小、预算、产能、供需、分配、运输（线性）、容量
- 【创新方向】约束工程化，验证与可解释

## 2. 核心原理（AI 极简版）

把目标函数与约束都写成线性形式，在可行域内找到使目标最优的解（全局最优、可解释）。

## 3. 美赛适用场景

- **典型任务**：生产/配给/预算分摊、线性运输、线性投资组合、线性成本最小化。
- **数据条件**：变量可连续化；约束能写成 \(A x \le b\) 或 \(A x = b\)。
- **Python 实现入口**：`scipy.optimize.linprog(method="highs")`。

## 4. 美赛创新适配方案（可直接用）

1) 【约束工程化】把现实规则显式写成约束（容量、最低服务、优先级、政策红线），提升“可落地性”。  
2) 【融合】用预测模型（ARIMA/Prophet/XGBoost）预测需求，再把预测值作为 LP 输入做“预测-优化”闭环。  
3) 【验证与可解释】输出影子价格/约束紧性（哪些约束在“卡脖子”），并做 ±10% 参数扰动灵敏度分析。

## 5. 优缺点与避坑指南

- **优点**：可解释、可复现、求解快；评审认可度高（基线稳）。
- **缺点**：真实问题常有非线性/整数决策，LP 可能“过度简化”。
- **避坑**：不要把整数/0-1 决策硬当连续变量；若必须离散，切换 MILP 或启发式算法。

## 6. 替代模型推荐

- MILP/0-1：存在选/不选、开/关、排班等离散决策时。
- NLP：目标或约束非线性时。
- GA/SA：规模过大或强非凸/强离散导致精确规划困难时。
