# 混合整数线性规划（MILP）/ 0-1 规划_优化类_MCM-B/ICM-D

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-B，ICM-D
- 【任务类型】选址调度，资源分配（离散），组合决策
- 【数据特征】离散决策变量（整数/0-1），约束可线性化
- 【触发关键词】选址、排班、开关决策、覆盖、路径选择（离散）、0-1、整数、容量约束
- 【创新方向】约束工程化，多目标与帕累托

## 2. 核心原理（AI 极简版）

在线性目标/约束下加入“整数/0-1”约束，使模型能表达“选或不选、开或不开、派或不派”的真实决策。

## 3. 美赛适用场景

- **典型任务**：设施选址、人员/车辆排班、离散资源分配、覆盖选点。
- **数据条件**：约束主要可线性表达；规模中小更稳（变量过大可能算不动）。
- **Python 实现入口**：`scipy.optimize.milp`；或用 `pulp` 建模（更易写约束）。

## 4. 美赛创新适配方案（可直接用）

1) 【多目标与帕累托】成本/公平/风险多目标：用加权法做基线，再用 NSGA-II 找帕累托前沿。  
2) 【分解-加速】先用贪心/聚类缩小候选集合，再做 MILP 精确求解（速度显著提升）。  
3) 【鲁棒/不确定性】对需求/成本做场景集（多场景约束）或蒙特卡罗评估，报告“方案稳定性”。

## 5. 优缺点与避坑指南

- **优点**：表达力强；决策变量“可落地”；评审喜欢“规则写进模型”。
- **缺点**：NP 难，规模大时求解慢甚至失败。
- **避坑**：不要把可线性化的非线性项漏掉；对大规模问题要准备启发式备选（GA/SA）。

## 6. 替代模型推荐

- LP：无整数/0-1 决策时。
- GA/SA/PSO：规模太大、约束复杂、求解时间受限时。
- CVXPY：当问题可写为凸优化（更稳、更强的求解器生态）。
