# 非线性规划（NLP，SLSQP 等）_优化类_MCM-A/ICM-D

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-A，ICM-D
- 【任务类型】参数反演/校准，工程设计优化，带约束最优化
- 【数据特征】连续决策变量；目标/约束含非线性项
- 【触发关键词】非线性、约束优化、参数估计、最小化误差、校准、设计变量、连续优化
- 【创新方向】约束工程化，验证与可解释

## 2. 核心原理（AI 极简版）

当目标或约束是非线性的，通过迭代搜索在约束可行域内找到局部最优（良好初始化很关键）。

## 3. 美赛适用场景

- **典型任务**：最小二乘参数反演、形状/几何优化、成本-性能非线性权衡。
- **数据条件**：可给出合理初值与边界；可计算目标函数（必要时可数值近似梯度）。
- **Python 实现入口**：`scipy.optimize.minimize(method="SLSQP")`（支持约束），或 `BFGS`（无约束/弱约束）。

## 4. 美赛创新适配方案（可直接用）

1) 【多起点 + 稳健性】多随机初值/多起点运行，报告最优值分布（避免“卡在局部最优”质疑）。  
2) 【约束工程化】把现实规则写成约束（上限/下限、平滑性、物理守恒/单调性）。  
3) 【敏感性 + 置信区间】对关键参数做 ±10% 扰动，并用自助法/蒙特卡罗估计输出区间。

## 5. 优缺点与避坑指南

- **优点**：能处理真实非线性；实现成本低；与机理模型/拟合天然兼容。
- **缺点**：可能只找到局部最优；对初值敏感。
- **避坑**：没有边界/初值就硬跑；若目标不可导/强噪声，优先考虑启发式（PSO/GA/SA）。

## 6. 替代模型推荐

- CVXPY：可转成凸优化时更稳。
- PSO/GA/SA：黑箱目标、强非凸、梯度不可用时。
- 曲线拟合：如果本质是“拟合参数”，可用 `curve_fit` 直接做。
