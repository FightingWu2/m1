# 凸优化（CVXPY）_优化类_ICM-D/MCM-A

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】ICM-D，MCM-A
- 【任务类型】资源分配（凸），投资组合，带复杂约束的工程优化
- 【数据特征】连续决策变量；问题可表述为凸（或近似凸）
- 【触发关键词】凸优化、二次规划、范数约束、稳健、约束多且复杂、可证明最优
- 【创新方向】约束工程化，鲁棒/不确定性

## 2. 核心原理（AI 极简版）

把问题写成凸目标 + 凸约束，利用凸性保证全局最优与求解稳定性；建模表达力通常强于“手写 Scipy”。

## 3. 美赛适用场景

- **典型任务**：二次规划（风险-收益）、范数正则（稀疏/平滑）、大量线性/锥约束的规划问题。
- **数据条件**：能判断/构造凸形式（必要时做合理凸近似）。
- **Python 实现入口**：`cvxpy`（`cp.Variable / cp.Minimize / cp.Problem.solve()`）。

## 4. 美赛创新适配方案（可直接用）

1) 【鲁棒优化】把不确定参数写成区间/场景，构造稳健约束（评审非常吃这一套）。  
2) 【二次项建模】用二次项表达“波动/风险/偏离惩罚”，比纯线性更贴近现实。  
3) 【可解释输出】报告活跃约束、对偶变量（影子价格）并做灵敏度分析。

## 5. 优缺点与避坑指南

- **优点**：建模强、求解稳；适合“复杂约束但仍可精确求解”的 ICM-D。
- **缺点**：要求问题（或近似）凸；不当的非凸写法会导致不可解或错误结论。
- **避坑**：先做“凸性检查”；非凸部分要么改写、要么切换到启发式/局部优化。

## 6. 替代模型推荐

- LP/MILP：纯线性或带整数约束时。
- NLP（SLSQP）：可接受局部最优且非凸时。
- GA/PSO：黑箱/强非凸/不可导时。
