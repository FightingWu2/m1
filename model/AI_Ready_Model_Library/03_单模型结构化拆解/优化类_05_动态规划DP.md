# 动态规划（DP）_优化类_MCM-B

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-B
- 【任务类型】多阶段决策，最短路（DP版），背包/分配
- 【数据特征】离散状态；满足最优子结构与无后效性
- 【触发关键词】多阶段、递推、状态、背包、阶段决策、子问题、最优子结构
- 【创新方向】约束工程化，验证与可解释

## 2. 核心原理（AI 极简版）

把原问题拆成可复用的子问题，用递推/表格自底向上求解，避免指数级重复计算。

## 3. 美赛适用场景

- **典型任务**：背包、阶段资源分配、某些 TSP/最短路的 DP 版本（规模较小时）。
- **数据条件**：状态空间不能太大（否则爆内存/爆时间）。
- **Python 实现入口**：自写 DP（`memoization` 或表格填充）；必要时配合 `numpy` 加速。

## 4. 美赛创新适配方案（可直接用）

1) 【状态压缩】用 bitmask/滚动数组减少内存；把“能压缩的维度”压掉。  
2) 【分解混合】大规模时先用启发式给近似解，再用 DP 在局部子集上精修。  
3) 【解释性】展示递推关系与状态转移图（评审理解成本极低）。

## 5. 优缺点与避坑指南

- **优点**：思路清晰、可解释；适合离散小中规模最优解。
- **缺点**：状态空间大时不可用（典型“维度灾难”）。
- **避坑**：先估算状态数；超过 1e7 级别基本就要换思路（启发式/分解）。

## 6. 替代模型推荐

- 最短路（Dijkstra/A*）：图上路径问题更通用。
- GA/SA：大规模组合优化的常用替代。
- MILP：能线性化且规模可控时可求精确解。
