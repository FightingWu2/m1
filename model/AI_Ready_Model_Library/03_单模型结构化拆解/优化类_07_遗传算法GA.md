# 遗传算法（GA）_优化类_MCM-B/ICM-D

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-B，ICM-D
- 【任务类型】组合优化，选址调度，TSP/路径，黑箱寻优
- 【数据特征】离散/混合变量；目标可能非凸/不可导
- 【触发关键词】组合优化、TSP、排班、选址、路径、全局搜索、交叉、变异、适应度
- 【创新方向】参数改进，多目标与帕累托

## 2. 核心原理（AI 极简版）

把解编码成“个体”，通过选择-交叉-变异迭代进化，逼近全局最优/近似最优。

## 3. 美赛适用场景

- **典型任务**：TSP/车辆路径、排班、选址、复杂约束的离散优化。
- **数据条件**：能定义可计算的适应度函数；约束可用惩罚项或修复算子处理。
- **Python 实现入口**：`deap` / `pygad`；或 `scikit-opt`。

## 4. 美赛创新适配方案（可直接用）

1) 【约束修复算子】对不可行解做“修复”（比纯惩罚更稳），并在论文中说明规则。  
2) 【混合算法】GA + SA（后期退火精修）或 GA + 局部搜索（2-opt/3-opt），防止早熟。  
3) 【多目标 GA】直接用 NSGA-II 输出帕累托解集（成本/时间/公平多目标）。

## 5. 优缺点与避坑指南

- **优点**：对离散问题鲁棒；不依赖梯度；实现灵活。
- **缺点**：调参敏感；算力消耗可能大；不保证最优。
- **避坑**：必须给出终止条件、种群规模、迭代次数，并做稳定性测试（多次运行取统计量）。

## 6. 替代模型推荐

- MILP：规模可控且可线性化时可求精确解。
- SA：需要更简单的单解搜索时。
- ACO：路径类问题常有更直观的叙事。
