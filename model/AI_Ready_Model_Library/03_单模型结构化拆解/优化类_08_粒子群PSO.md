# 粒子群优化（PSO）_优化类_MCM-A/ICM-D

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-A，ICM-D
- 【任务类型】连续变量全局寻优，参数反演，黑箱优化
- 【数据特征】连续决策变量；目标可计算但可能不可导/强非凸
- 【触发关键词】连续优化、参数寻优、全局搜索、速度位置更新、惯性权重
- 【创新方向】参数改进，鲁棒/不确定性

## 2. 核心原理（AI 极简版）

用一群粒子在搜索空间中飞行，结合个体最优与群体最优引导搜索，快速找到高质量解。

## 3. 美赛适用场景

- **典型任务**：连续参数标定、非凸函数最小化、模型校准（不想推导梯度）。
- **数据条件**：能给出参数边界；适应度可计算。
- **Python 实现入口**：`pyswarm` 或 `scikit-opt`（PSO）。

## 4. 美赛创新适配方案（可直接用）

1) 【自适应参数】惯性权重/学习因子随迭代衰减，提高收敛与跳出局部最优能力。  
2) 【多启动 + 统计】多次运行取均值/最优/方差，并报告稳定性（评审会看）。  
3) 【融合不确定性】在适应度中加入置信区间/风险惩罚（如 CVaR 风格惩罚）。

## 5. 优缺点与避坑指南

- **优点**：实现简单；对连续变量收敛快；不需梯度。
- **缺点**：易早熟；高维时性能下降。
- **避坑**：不给边界就跑；没有多次运行统计就直接报一个结果。

## 6. 替代模型推荐

- NLP（SLSQP）：目标可微且约束严格时。
- GA：离散/混合变量更合适。
- SA：单解搜索更轻量。
