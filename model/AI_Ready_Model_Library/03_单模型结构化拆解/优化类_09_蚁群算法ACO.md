# 蚁群算法（ACO）_优化类_MCM-B

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-B
- 【任务类型】路径规划，TSP/VRP 类组合优化
- 【数据特征】图结构；边权可定义（距离/时间/成本）
- 【触发关键词】路径、TSP、车辆路径、信息素、概率选边、迭代更新
- 【创新方向】参数改进，约束工程化

## 2. 核心原理（AI 极简版）

通过“信息素”强化优质路径，利用概率式搜索在图上迭代构造路径，逐步逼近最优路线方案。

## 3. 美赛适用场景

- **典型任务**：TSP、物流路径、旅行线路规划、网络路由。
- **数据条件**：可构造距离/时间矩阵；能处理约束（容量/时间窗可用惩罚或修复）。
- **Python 实现入口**：通常自写（规则清晰利于论文表达）；也可基于通用元启发式框架封装。

## 4. 美赛创新适配方案（可直接用）

1) 【信息素蒸发自适应】随迭代调整蒸发率，平衡探索与利用。  
2) 【启发式初始化】用最近邻/贪心生成初解，再由 ACO 强化（收敛更快）。  
3) 【约束修复】时间窗/容量用修复算子保证可行（比惩罚项更稳定）。

## 5. 优缺点与避坑指南

- **优点**：路径类叙事直观；容易画图展示改进过程；适合 MCM-B。
- **缺点**：参数多；大规模时计算量大。
- **避坑**：必须展示收敛曲线/多次运行统计；否则评审会认为“随机性太强”。

## 6. 替代模型推荐

- GA/SA：路径类通用替代，生态更成熟。
- Dijkstra/A*：如果是“单源最短路”而不是 TSP。
- MILP：规模可控且能线性化时可求精确解。
