# 人工蜂群算法（ABC）_优化类_MCM-A/ICM-D

## 1. 模型基础信息

- 【模型类别】优化类
- 【题型适配】MCM-A，ICM-D
- 【任务类型】连续域全局优化，多峰函数寻优
- 【数据特征】连续决策变量；目标函数多峰/非凸
- 【触发关键词】多峰、全局寻优、雇佣蜂、跟随蜂、侦察蜂、随机探索
- 【创新方向】参数改进，鲁棒/不确定性

## 2. 核心原理（AI 极简版）

用三类“蜂”在解空间中探索与开发：发现优质食源并在其附近强化搜索，同时保持随机探索避免陷入局部最优。

## 3. 美赛适用场景

- **典型任务**：连续参数寻优、黑箱目标、多峰函数优化（PSO/GA 的备选）。
- **数据条件**：可给定参数边界；适应度可计算。
- **Python 实现入口**：一般自写实现（便于解释）；或用通用智能算法库封装。

## 4. 美赛创新适配方案（可直接用）

1) 【对比实验】与 PSO/GA/SA 做同算力对比（收敛速度、最优值、稳定性），这在评审眼里是“研究味”。  
2) 【自适应搜索】侦察概率/邻域步长随迭代衰减，提升后期精修能力。  
3) 【不确定性惩罚】把风险/波动加入适应度，输出更稳健的参数组合。

## 5. 优缺点与避坑指南

- **优点**：对多峰问题表现常不错；探索性强。
- **缺点**：实现细节决定效果；缺少标准库时需要自写。
- **避坑**：不要把“随机搜索”包装成 ABC；必须说明三类蜂的更新规则与参数含义。

## 6. 替代模型推荐

- PSO：连续优化更常用、生态更成熟。
- GA：混合/离散变量更适合。
- NLP（SLSQP）：可微且约束明确时更稳、更快。
