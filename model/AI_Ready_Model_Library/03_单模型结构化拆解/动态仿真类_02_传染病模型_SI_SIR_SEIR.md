# 传染病模型（SI/SIS/SIR/SEIR + 改进项）_动态仿真类_MCM-A/ICM-E

## 1. 模型基础信息

- 【模型类别】动态仿真类
- 【题型适配】MCM-A，ICM-E
- 【任务类型】传播建模，干预评估，预测与情景模拟
- 【数据特征】人群分舱（易感/感染/康复/潜伏）；时间序列观测
- 【触发关键词】传播、感染率、接触率、隔离、再生数、SIR、SEIR
- 【创新方向】约束工程化，鲁棒/不确定性

## 2. 核心原理（AI 极简版）

把人群按状态分舱，用微分方程描述舱室之间的转移；通过参数变化模拟干预（隔离、疫苗、社交距离）的效果。

## 3. 美赛适用场景

- **典型任务**：疾病/信息/舆情传播；干预策略对峰值、持续时间、总规模的影响评估。
- **数据条件**：需要病例/传播数据；参数可拟合；可加入时变接触率反映政策变化。
- **Python 实现入口**：`scipy.integrate.solve_ivp` + 参数拟合（`curve_fit`/`minimize`）。

## 4. 美赛创新适配方案（可直接用）

1) 【模型改进】SEIR-SD：加入社交距离/隔离参数，或加入时变 \(\beta(t)\) 表示政策强度。  
2) 【不确定性】对 \(\beta,\gamma\) 做区间/分布假设，用蒙特卡罗输出区间预测。  
3) 【校准与验证】分时段拟合（政策前后），并做滚动预测验证。

## 5. 优缺点与避坑指南

- **优点**：解释性强；干预评估写作非常友好；E 题高频。
- **缺点**：分舱假设简化；真实传播存在网络结构与异质性。
- **避坑**：别把“传播”都当 SIR；若传播依赖网络结构，需结合网络/ABM。

## 6. 替代模型推荐

- ABM：个体异质性、空间移动、网络传播更关键时。
- 马尔可夫链：离散状态转移、规则更粗粒度时。
- 纯时间序列（ARIMA/Prophet）：只要预测不做机理解释时。
