# 偏微分方程（PDE）数值解：FDM/FEM_动态仿真类_MCM-A/ICM-E

## 1. 模型基础信息

- 【模型类别】动态仿真类
- 【题型适配】MCM-A，ICM-E
- 【任务类型】空间-时间扩散/传播/热传导仿真
- 【数据特征】空间数据（网格/区域）+ 时间演化
- 【触发关键词】热传导、扩散、波动、PDE、空间分布、网格、边界条件
- 【创新方向】约束工程化，验证与可解释

## 2. 核心原理（AI 极简版）

PDE 同时描述时间与空间变化；用离散化（有限差分/有限元）把连续方程转成可计算的线性/非线性代数系统并迭代求解。

## 3. 美赛适用场景

- **典型任务**：污染扩散、热量分布、波动传播、空间传播过程（E 题高频）。
- **数据条件**：需要边界条件、初值与网格划分；扩散系数等参数可通过拟合估计。
- **Python 实现入口**：FDM 可自写 `numpy` 网格迭代；FEM 可用 `FEniCS`（环境允许时）。

## 4. 美赛创新适配方案（可直接用）

1) 【网格收敛】做网格加密对比，展示数值解收敛（非常像“工程论文”，加分）。  
2) 【参数反演】用观测点数据拟合扩散系数/源项强度（优化 + PDE）。  
3) 【情景模拟】不同风向/边界/治理强度下的扩散对比，输出空间热力图。

## 5. 优缺点与避坑指南

- **优点**：空间过程解释力强；图表表现极佳；适合 A/E 题冲高分。
- **缺点**：实现成本较高；数值稳定性/边界处理容易出错。
- **避坑**：必须说明离散格式与稳定性条件（至少定性说明）；不要用太细网格导致算不完。

## 6. 替代模型推荐

- ODE：无空间维度时。
- CA：更偏规则驱动的离散空间扩散（更轻量）。
- PINNs：希望用“机理+AI”作为创新点时。
