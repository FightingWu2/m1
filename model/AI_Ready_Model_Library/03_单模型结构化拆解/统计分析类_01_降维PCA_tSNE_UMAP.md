# 降维（PCA / t-SNE / UMAP）_统计分析类_MCM-C

## 1. 模型基础信息

- 【模型类别】统计分析类
- 【题型适配】MCM-C
- 【任务类型】数据预处理，特征压缩，可视化洞察
- 【数据特征】高维数据；共线性强；需可视化/聚类前处理
- 【触发关键词】降维、共线性、主成分、可视化、t-SNE、UMAP
- 【创新方向】验证与可解释，模型融合

## 2. 核心原理（AI 极简版）

PCA 用线性变换保留最大方差方向；t-SNE/UMAP 用非线性流形方法把高维结构压到 2D/3D，便于展示聚类与结构。

## 3. 美赛适用场景

- **典型任务**：指标太多、相关性强；需要画出“分群结构/异常点”。
- **数据条件**：特征需标准化；t-SNE/UMAP 用于可视化为主（不宜直接用于预测）。
- **Python 实现入口**：`sklearn.decomposition.PCA`；`sklearn.manifold.TSNE`；UMAP（环境允许时 `umap-learn`）。

## 4. 美赛创新适配方案（可直接用）

1) 【PCA-DEA/PCA-回归】先 PCA 去共线降维，再做 DEA/回归/聚类，提高稳健性。  
2) 【解释主成分】给出载荷矩阵与解释方差比，把“主成分含义”写成可读结论。  
3) 【可视化叙事】用 2D/3D 可视化支持分群、异常、政策分区等论点。

## 5. 优缺点与避坑指南

- **优点**：降维后模型更稳；图表更漂亮；评审容易理解。
- **缺点**：t-SNE/UMAP 有随机性；参数不同图形不同。
- **避坑**：可视化必须固定随机种子；不要用 t-SNE 结果做严肃的距离/回归推断。

## 6. 替代模型推荐

- 因子分析：更强调潜变量结构时。
- 特征选择（XGBoost重要性）：更偏监督学习的筛特征。
- 相关分析：只想看相关性而非降维时。
