# 相关分析（Pearson / Spearman）_统计分析类_MCM-C

## 1. 模型基础信息

- 【模型类别】统计分析类
- 【题型适配】MCM-C
- 【任务类型】因果/关系（相关性探索），特征筛选
- 【数据特征】连续/序数变量；可能非正态（Spearman 更稳）
- 【触发关键词】相关系数、显著性、线性相关、秩相关、关系强弱
- 【创新方向】验证与可解释

## 2. 核心原理（AI 极简版）

Pearson 衡量线性相关，Spearman 衡量单调相关（基于秩，对非正态与异常值更稳），并可做显著性检验。

## 3. 美赛适用场景

- **典型任务**：探索变量关系、筛选候选特征、支撑后续回归/ML 建模解释。
- **数据条件**：注意异常值与缺失值处理；相关不等于因果，需要文字说明边界。
- **Python 实现入口**：`scipy.stats.pearsonr/spearmanr` 或 `pandas.DataFrame.corr`。

## 4. 美赛创新适配方案（可直接用）

1) 【相关矩阵可视化】热力图 + 聚类排序，快速展示结构。  
2) 【稳健对比】Pearson vs Spearman 对比，说明数据分布与异常点影响。  
3) 【与模型融合】用相关筛特征 + 回归/XGBoost 建模，再用 SHAP/系数解释“影响路径”。

## 5. 优缺点与避坑指南

- **优点**：快速、直观；写作门槛低；评审看得懂。
- **缺点**：只能说明相关，不代表因果；多重比较会产生伪相关。
- **避坑**：不要用相关直接下因果结论；变量多时要说明多重检验或只做探索性分析。

## 6. 替代模型推荐

- CCA：两组变量整体相关关系时。
- SEM：需要“路径/潜变量”的结构解释时。
- GRA：小样本、不完备信息时。
