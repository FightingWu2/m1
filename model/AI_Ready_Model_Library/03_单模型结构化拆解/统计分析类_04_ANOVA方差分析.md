# 方差分析（ANOVA）_统计分析类_MCM-C/ICM-F

## 1. 模型基础信息

- 【模型类别】统计分析类
- 【题型适配】MCM-C，ICM-F
- 【任务类型】组间差异检验，政策/实验效果评估
- 【数据特征】分组数据；关注均值差异的显著性
- 【触发关键词】显著性、组间差异、实验设计、政策效果、ANOVA
- 【创新方向】验证与可解释

## 2. 核心原理（AI 极简版）

把总变异分解为“组间变异”和“组内变异”，用 F 检验判断不同组均值差异是否显著。

## 3. 美赛适用场景

- **典型任务**：比较不同策略/区域/人群的指标差异；支持“政策是否有效”的结论。
- **数据条件**：独立性与方差齐性假设要说明；必要时用非参数替代。
- **Python 实现入口**：`scipy.stats.f_oneway`；更复杂设计用 `statsmodels.stats.anova`。

## 4. 美赛创新适配方案（可直接用）

1) 【事后检验】显著后做多重比较（Tukey 等），定位差异来自哪些组。  
2) 【效应量】报告效应量（不仅 p 值），提升论文质量。  
3) 【与回归/SEM融合】把显著差异转成解释变量，进入回归或结构模型做机制解释。

## 5. 优缺点与避坑指南

- **优点**：统计推断严谨；适合“对比结论”。
- **缺点**：假设不满足时结论不可靠。
- **避坑**：别只写 p 值；必须说明假设检查；样本量过小慎用。

## 6. 替代模型推荐

- Kruskal-Wallis：非正态或方差不齐时的非参数替代。
- 回归：想控制混杂因素并解释影响时。
- 蒙特卡罗/Bootstrap：想要区间而非显著性时。
