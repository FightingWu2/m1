# 判别分析（LDA/QDA）_统计分析类_MCM-C

## 1. 模型基础信息

- 【模型类别】统计分析类
- 【题型适配】MCM-C
- 【任务类型】分类（可解释），特征降维（判别轴）
- 【数据特征】有标签分类；类内分布近似高斯（LDA/QDA 的经典假设）
- 【触发关键词】判别、分类、线性判别、二次判别、可解释边界
- 【创新方向】验证与可解释

## 2. 核心原理（AI 极简版）

在高斯分布假设下推导最优判别函数：LDA 假设各类协方差相同得到线性边界，QDA 允许协方差不同得到二次边界。

## 3. 美赛适用场景

- **典型任务**：风险等级分类、是否达标判别、小样本分类基线。
- **数据条件**：特征标准化；类别不平衡需处理；最好报告混淆矩阵。
- **Python 实现入口**：`sklearn.discriminant_analysis.LinearDiscriminantAnalysis/QuadraticDiscriminantAnalysis`。

## 4. 美赛创新适配方案（可直接用）

1) 【可解释分类】展示判别系数/判别轴方向，并解释哪些特征推动分类。  
2) 【与PCA融合】先 PCA 去噪，再 LDA 做判别，提升稳定性。  
3) 【严格验证】k-fold CV + 混淆矩阵 + ROC/AUC（若二分类）形成完整证据链。

## 5. 优缺点与避坑指南

- **优点**：解释性强；小样本基线好写。
- **缺点**：分布假设不满足时性能下降。
- **避坑**：不要忽略类别不平衡；不要只报 accuracy；要给精确率/召回率。

## 6. 替代模型推荐

- SVM：非线性边界更强时。
- 随机森林/XGBoost：追求精度与非线性表达时。
- 逻辑回归：更简单且有概率解释时。
