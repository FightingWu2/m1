# 蒙特卡罗模拟（Monte Carlo）_随机过程类_全题型通用

## 1. 模型基础信息

- 【模型类别】随机过程类
- 【题型适配】全题型通用
- 【任务类型】风险不确定，仿真复现，数值积分
- 【数据特征】存在随机性/不确定参数；可采样生成随机输入
- 【触发关键词】不确定性、风险、随机抽样、置信区间、概率估计、场景模拟
- 【创新方向】鲁棒/不确定性，验证与可解释

## 2. 核心原理（AI 极简版）

对不确定变量进行大量随机抽样，重复计算目标结果，用样本分布近似真实分布与风险区间。

## 3. 美赛适用场景

- **典型任务**：风险评估、稳健性评估、区间预测、随机过程的数值近似。
- **数据条件**：需要定义随机变量分布（来自数据拟合或合理假设并解释）。
- **Python 实现入口**：`numpy.random` + `scipy.stats`（分布拟合/采样）。

## 4. 美赛创新适配方案（可直接用）

1) 【区间结果】报告均值/分位数/置信区间，并给“保守策略”建议（评审最爱）。  
2) 【敏感性结合】对关键分布参数做扰动，输出风险随参数变化曲线。  
3) 【与优化融合】用 MC 评估候选方案风险，再在“期望收益-风险”上做多目标权衡。

## 5. 优缺点与避坑指南

- **优点**：概念简单；适用面极广；写作加分明显（风险意识）。
- **缺点**：计算量大；分布假设不当会误导。
- **避坑**：必须说明分布来源；必须给抽样次数与收敛性；不要用太少样本就下结论。

## 6. 替代模型推荐

- GPR：想要模型自带不确定性区间时。
- Sobol/Morris：更系统的全局敏感性分析时。
- 场景分析：时间紧且只需少量情景时。
