# ARIMA/SARIMA_预测类_MCM-C/ICM-E/ICM-F

## 1. 模型基础信息

- 【模型类别】预测类
- 【题型适配】MCM-C，ICM-E，ICM-F
- 【任务类型】预测回归（时间序列），趋势外推
- 【数据特征】时序数据；平稳或可差分平稳；可能有季节性（SARIMA）
- 【触发关键词】时间序列、平稳、差分、自相关、季节性、短期预测
- 【创新方向】验证与可解释，模型融合

## 2. 核心原理（AI 极简版）

用“差分 + 自回归 + 移动平均”刻画时间序列的自相关结构，对未来做统计外推；有季节性时用 SARIMA。

## 3. 美赛适用场景

- **典型任务**：短期需求/流量/价格等预测；基线预测模型。
- **数据条件**：样本量>30 更稳；需做平稳性检验（ADF）与残差检验。
- **Python 实现入口**：`statsmodels.tsa.arima.model.ARIMA` / `statsmodels.tsa.statespace.SARIMAX`。

## 4. 美赛创新适配方案（可直接用）

1) 【模型诊断】ADF/ACF/PACF + 残差白噪声检验，展示“统计严谨”。  
2) 【融合】ARIMA + LSTM/Prophet 加权集成，提升精度并给出对比表。  
3) 【不确定性】输出预测区间（置信区间）并解释风险含义。

## 5. 优缺点与避坑指南

- **优点**：强基线；解释性强；评审认可。
- **缺点**：对非线性强/结构突变的序列效果有限。
- **避坑**：不要跳过平稳性检验；不要把明显受政策/节假日影响的数据硬塞 ARIMA。

## 6. 替代模型推荐

- 指数平滑：趋势/季节性更“工程化”。
- Prophet：缺失值多、节假日效应强时更稳。
- LSTM/GRU：数据量大、非线性强时更优。
