# 指数平滑（Holt / Holt-Winters）_预测类_MCM-C/ICM-E/ICM-F

## 1. 模型基础信息

- 【模型类别】预测类
- 【题型适配】MCM-C，ICM-E，ICM-F
- 【任务类型】预测回归（时间序列），趋势/季节性预测
- 【数据特征】时序数据；存在趋势与/或季节性
- 【触发关键词】指数平滑、趋势、季节性、Holt、Holt-Winters、平滑系数
- 【创新方向】验证与可解释，模型融合

## 2. 核心原理（AI 极简版）

用递推式对水平项、趋势项、季节项做指数加权更新，得到平滑的趋势外推。

## 3. 美赛适用场景

- **典型任务**：业务量/能耗/客流等有趋势或季节性的预测；比 ARIMA 更好“讲清楚”。
- **数据条件**：季节周期明确；异常点需处理（否则平滑被破坏）。
- **Python 实现入口**：`statsmodels.tsa.holtwinters.ExponentialSmoothing`。

## 4. 美赛创新适配方案（可直接用）

1) 【参数选择对比】对不同季节周期/趋势形式（加性/乘性）做对比，给出最优配置。  
2) 【异常点处理】先做异常检测/截尾，再做指数平滑，提高鲁棒性。  
3) 【集成】与 ARIMA/Prophet 做集成，输出“稳健预测”。

## 5. 优缺点与避坑指南

- **优点**：实现简单；对趋势/季节性直观；适合快速基线。
- **缺点**：对结构突变不敏感；长周期复杂季节性不如 Prophet。
- **避坑**：不要忽略季节周期；不要让异常点主导平滑参数。

## 6. 替代模型推荐

- ARIMA/SARIMA：需要统计检验与残差诊断时。
- Prophet：节假日效应、缺失值多时。
- LSTM/GRU：非线性强、数据量大时。
