# 回归分析家族（线性/多项式/Ridge/Lasso/ElasticNet）_预测类_MCM-C/ICM-F

## 1. 模型基础信息

- 【模型类别】预测类
- 【题型适配】MCM-C，ICM-F
- 【任务类型】预测回归，关系建模（可解释）
- 【数据特征】结构化表格数据；可能存在多重共线性（正则化可缓解）
- 【触发关键词】回归、因子影响、可解释、系数、共线性、正则化、预测
- 【创新方向】验证与可解释，参数改进

## 2. 核心原理（AI 极简版）

用线性（或多项式扩展）表达自变量与目标的关系；Ridge/Lasso/ElasticNet 通过正则化抑制过拟合并缓解共线性。

## 3. 美赛适用场景

- **典型任务**：解释“哪些因素影响目标”；给出可解释预测与政策建议（ICM-F 高适配）。
- **数据条件**：特征需要标准化（正则化模型尤其需要）；异常值需处理。
- **Python 实现入口**：`sklearn.linear_model.LinearRegression/Ridge/Lasso/ElasticNet`；统计推断可用 `statsmodels`。

## 4. 美赛创新适配方案（可直接用）

1) 【可解释性】报告系数、显著性（用 `statsmodels`）、置信区间，并给政策含义。  
2) 【交叉验证选参】用 `GridSearchCV` 选择正则化强度，给出验证误差曲线。  
3) 【特征工程】引入交互项/多项式项（但要做消融证明必要性）。

## 5. 优缺点与避坑指南

- **优点**：解释性强；写作友好；可作为所有 ML 的基线。
- **缺点**：对强非线性关系表达有限。
- **避坑**：不要只报 R²；要做训练/验证划分与残差诊断；共线性严重时别用“裸线性回归”。

## 6. 替代模型推荐

- XGBoost/随机森林：非线性强且追求精度时。
- GPR：需要不确定性区间并且数据量不大时。
- 因子分析/SEM：更偏“潜变量/结构路径”解释时。
