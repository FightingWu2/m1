# 支持向量机（SVR/SVM）_预测类_MCM-C

## 1. 模型基础信息

- 【模型类别】预测类
- 【题型适配】MCM-C
- 【任务类型】预测回归（SVR），分类（SVM）
- 【数据特征】中小样本；非线性关系可用核函数
- 【触发关键词】小样本、核函数、间隔最大化、回归、分类、非线性
- 【创新方向】参数改进，验证与可解释

## 2. 核心原理（AI 极简版）

通过最大化间隔获得泛化能力；用核函数把非线性问题映射到高维线性可分空间（SVR 对应回归版本）。

## 3. 美赛适用场景

- **典型任务**：小样本回归预测、二分类判别（故障/风险/是否达标）。
- **数据条件**：特征需标准化；核函数与超参数需要调优。
- **Python 实现入口**：`sklearn.svm.SVR` / `sklearn.svm.SVC`。

## 4. 美赛创新适配方案（可直接用）

1) 【网格搜索】用 `GridSearchCV` 搜索 C/γ/核函数，并报告最优参数与验证结果。  
2) 【可解释补充】用特征敏感性（置换重要性）或局部解释方法解释关键因素。  
3) 【与PCA融合】先 PCA 去噪降维，再 SVR，提高稳健性。

## 5. 优缺点与避坑指南

- **优点**：小样本表现常好；非线性能力强。
- **缺点**：大样本训练慢；超参数敏感；可解释性弱于线性回归。
- **避坑**：必须标准化；必须交叉验证；不要把 SVM 当“万能模型”。

## 6. 替代模型推荐

- 随机森林/XGBoost：大样本或特征多时更稳。
- 线性/正则回归：强解释需求时更合适。
- GPR：需要预测区间时更加分。
