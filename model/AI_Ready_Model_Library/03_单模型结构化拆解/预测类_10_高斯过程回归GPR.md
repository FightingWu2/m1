# 高斯过程回归（GPR）_预测类_MCM-C/ICM-E/ICM-F

## 1. 模型基础信息

- 【模型类别】预测类
- 【题型适配】MCM-C，ICM-E，ICM-F
- 【任务类型】预测回归 + 不确定性量化（置信区间）
- 【数据特征】中小样本；希望同时给“点预测 + 置信区间”
- 【触发关键词】置信区间、不确定性、核函数、贝叶斯回归、小样本
- 【创新方向】鲁棒/不确定性，验证与可解释

## 2. 核心原理（AI 极简版）

把函数值视为高斯过程，实现贝叶斯回归：不仅预测均值，还能给出预测方差（不确定性范围）。

## 3. 美赛适用场景

- **典型任务**：评审特别喜欢的“区间预测/风险范围”；样本不大但需要可信区间。
- **数据条件**：特征需标准化；核函数选择影响大。
- **Python 实现入口**：`sklearn.gaussian_process.GaussianProcessRegressor` + 核函数（RBF/Matérn 等）。

## 4. 美赛创新适配方案（可直接用）

1) 【区间解释】把置信区间转成“风险/保守策略”建议（极好写、极加分）。  
2) 【核函数对比】RBF vs Matérn 对比，展示选择依据与效果。  
3) 【与优化融合】用 GPR 做代理模型（surrogate），再用于优化（减少昂贵仿真次数）。

## 5. 优缺点与避坑指南

- **优点**：自带不确定性；小样本表现好；写作表现力强。
- **缺点**：大样本训练慢；核函数选择敏感。
- **避坑**：别在 n>5000 场景硬用；必须说明核函数与超参数；不要误读区间为“必然范围”。

## 6. 替代模型推荐

- XGBoost：大样本高维场景更实用。
- Bootstrap + 任意回归：想要区间但不想用 GPR 时。
- 线性/正则回归：需要推断与解释时。
