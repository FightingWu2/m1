# 深度学习（MLP/CNN/LSTM/GRU）_预测类_MCM-C/ICM-E/ICM-F

## 1. 模型基础信息

- 【模型类别】预测类
- 【题型适配】MCM-C，ICM-E，ICM-F
- 【任务类型】预测回归，分类；时序预测（LSTM/GRU）；图像/网格模式（CNN）
- 【数据特征】中大样本更稳；非线性强；需要训练/验证流程
- 【触发关键词】深度学习、非线性、神经网络、LSTM、GRU、注意力、过拟合
- 【创新方向】参数改进，模型融合

## 2. 核心原理（AI 极简版）

用多层非线性函数逼近复杂映射；LSTM/GRU 用门控机制捕捉长时依赖；CNN 擅长空间/局部特征提取。

## 3. 美赛适用场景

- **典型任务**：大样本时序预测（LSTM/GRU）；复杂非线性预测；需要较高精度的回归/分类。
- **数据条件**：必须做标准化；必须做训练/验证切分；建议做早停与正则化防过拟合。
- **Python 实现入口**：`pytorch` / `tensorflow/keras`。

## 4. 美赛创新适配方案（可直接用）

1) 【注意力机制】在 LSTM/GRU 上加注意力，突出关键时间段（常见加分点）。  
2) 【融合】ARIMA/Prophet 负责可解释趋势，神经网络拟合残差（“强解释+强精度”）。  
3) 【可解释补充】用特征消融/SHAP（若可用）或注意力权重可视化，给出“为什么这么预测”。

## 5. 优缺点与避坑指南

- **优点**：表达力强；非线性预测能力突出；融合后易冲高分。
- **缺点**：样本不够时易过拟合；实现与训练成本较高；可解释性弱。
- **避坑**：小样本不要硬上深度学习；必须交叉验证或时间序列滚动验证；必须报告超参与训练细节。

## 6. 替代模型推荐

- XGBoost：结构化数据小中样本通常更稳且更易解释。
- ARIMA/指数平滑/Prophet：时间序列基线，写作更稳。
- GPR：小样本且要区间预测时。
